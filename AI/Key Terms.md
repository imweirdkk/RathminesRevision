# Turing Test

A Turing Test is a method of inquiry in Artificial Intelligence for determining a computer's ability to imitate a human. The test is named after British mathematician and logician, Alan Turing.

# Intelligent Agent (IA)

In AI, an intelligent agent is something that can make decision based on the inputs received from its environment. It perceives the environment through sensors and acts on those inputs through actuators.

# Expert System

It is a piece of software which uses inference engines and databases of expert knowledge to offer advice or make decisions in areas like medical diagnosis. An inference engine applies logical rules to generate new information.

# Fuzzy Logic

A form of many-valued logic in which the truth values of variables may be any real number between 0 and 1, both inclusive. It is used in situations where the truth value lies between completely true and completely false. 

# Bayesian Programming

This statistical approach uses probabilistic models to solve problems when less than the necessary information is available.

# Committee Machine 

A committee machine is a type of artificial neural network that uses a divide and conquer strategy to combine responses from multiple neural networks into a single form.

# Generative AI (GenAI)

Generative AI (GenAI) is a type of artificial intelligence that can create a wide variety of data, such as images, videos, audio, text, and 3D models. It does this by learning patters from existing data, then using this knowledge to generate new and unique outputs.

# Algorithm

An algorithm is a set of sequentially organized instructions for problem-solving operations. Algorithms are used in the field of mathematics and computer science

# Complexity 

Complexity can be defined as a mathematical characterization of the difficulty of a mathematical problem, describing the resources required by a computing machine to solve the problem

# Heuristics

It is a problem-solving technique used when classic methods are too clow, or for finding an approximate solution when classic methods fail to find any exact solution

# Machine Learning

Machine learning is a subset of AI; it utilizes computer algorithms that improve through experience

# Error-driven learning

This is a type of reinforcement learning technique within machine learning. Error-driven learning focuses on how an agent ought to take actions in an environment to bring down the amount of error-driven feedback

# Back propagation 

This widely used algorithm in training feeds back information about errors so that parameters, or characteristics, can be changed to make better decisions in the future.
