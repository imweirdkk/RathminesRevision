# The Machine Learning Process

Compared to algorithm-based functions, machine learning is relatively independent because it has the ability to learn from training examples, which are sets of relevant data. It identifies patterns in relationships and then comes up with valuable predictions as outputs. 

# The Key Characteristics of Machine Learning

Common implementations of machine learning include:
- Automation of repetitive tasks, resulting in improved productivity.
- Execution of accurate data analysis
- Automation of digital visualization
- Generation of predictable, transparent, and consistent outcomes

# Machine Learning for Businesses

Machine Learning is helping businesses boost their total efficiency. For instance, Sage, an accounting and payroll software company, uses machine learning to track down potential candidates and schedule interviews, saving time and effort. 

# Techniques within Machine Learning

Machine learning is a broad and complex area. Two important techniques it uses are "Error-driven Learning" & "Backpropagation"
