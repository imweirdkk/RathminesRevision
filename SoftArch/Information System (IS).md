# Definition of an Information System:
An information system takes data as input, processes it, and produces useful information as output to help organizations operate effectively

# Re-worded:
An information system takes raw data, processes it and outputs useful information for the organization

# Key Components of an Information System

**1. Hardware** - physical devices (computers, servers)
**2. Software** - applications and programs
**3. Data**  - raw facts that are processed into information
**4. People** - users and IT professionals
**5. Processes** - procedures and rules for using the system

# Types of Information Systems
    - Transaction Processing Systems (TPS)
    - Management Information Systems (MIS)
    - Decision Support Systems (DSS)
    - Executive Support Systems (ESS) / Executive Information Systems (EIS)
    - Office Automation Systems (OAS)
    - Knowledge Management Systems (KMS)
