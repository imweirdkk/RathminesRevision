**Definition:**
- It automates the assignment of IP addresses, subnet masks, and gateways

**Dynamic Leases:**
- Devices request an IP upon connecting, and the DHCP server "leases" them an address temporarily
- I.E: (Assuming a Network/24) When your phone connects it'll be known as 192; the next time it connects it may be known as 134 since another device is currently borrowing the name 192

**Use Cases:**
- Ideal for environments with many end-user devices (laptops, smartphones, guest Wi-Fi) where manual configuration is impractical
