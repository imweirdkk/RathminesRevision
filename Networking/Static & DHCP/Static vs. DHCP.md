
# Comparison 

| Feature       | [[Static IP]]                                    | [[DHCP - Dynamic Host Configuration Protocol\|DHCP (Dynamic)]] |
| ------------- | ------------------------------------------------ | -------------------------------------------------------------- |
| Assignment    | Manual (by Admin)                                | Automatic (by Server)                                          |
| Duration      | Permanent (Fixed)                                | Temporary (Leased)                                             |
| Overhead      | High (Hard to scale)                             | Low (East to scale)                                            |
| Best Use Case | - Servers<br>- Printers<br>- Routers<br>- etc... | - Laptops<br>- Phones<br>- Guests<br>- etc                     |
